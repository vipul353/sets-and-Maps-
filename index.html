Q1. "Problem: You are given a string (STR) of length N, consisting of only the lower case English alphabet. Your task is to remove all the duplicate occurrences of characters in the string. Input: abcadeecfb Output: abcdef"

Ans:-

        function removeDuplicates(str) {
            const uniqueChars = [...new Set(str)];
            return uniqueChars.join('');
          }
          
          const input = "abcadeecfb";
          const output = removeDuplicates(input);
          console.log(output);

Output:-"abcdef"

explanation:-

The removeDuplicates function takes a string str as input. It converts the string to a set using the spread operator and the Set constructor, which removes duplicate characters. Then, it converts the unique characters back to a string using join. The function returns the resulting string without duplicate occurrences. In the provided example, the input string "abcadeecfb" is processed, and the output is "abcdef", which is the input string without any duplicate characters

Q2. "Problem: You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps) Input: abcadeecfb Output: a=2 b=2 c=2 d=1 e=2 f=1".

Ans:-

        function countAlphabets(str) {
            const charCountMap = new Map();
            for (let i = 0; i < str.length; i++) {
              const char = str[i];
              if (charCountMap.has(char)) {
                charCountMap.set(char, charCountMap.get(char) + 1);
              } else {
                charCountMap.set(char, 1);
              }
            }
            for (const [char, count] of charCountMap) {
              console.log(`${char}=${count}`);
            }
          }
          
          const input = "abcadeecfb";
          countAlphabets(input);

Output:-a=2 b=2 c=2 d=1 e=2 f=1

explanation:-

The countAlphabets function takes a string str as input. It initializes a new Map called charCountMap to store the counts of each alphabet. It iterates through each character in the string, updating the count in the map accordingly. Finally, it loops through the charCountMap and prints the count for each alphabet
